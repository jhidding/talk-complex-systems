## What is a complex system?

<ul>
<li> Simple microscopic rules of interaction </li>
<li class="fragment"> Emergent large scale behaviour:

- scaling laws
- pattern formation
- synchronisation
- self-replication
- co-evolution
- non-linear / chaotic
- robustness
- path-dependence
</li></ul>
